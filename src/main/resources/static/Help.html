帮助文档

                            欢迎使用我们的网站！接下来我将带您了解每个部分的功能以及如何使用。

1. Home
Home 是访问我们网站的入口

2. Model
Model 简要的介绍了我们的模型 —— ImmunoFlash

3. Web Server
Web Server 提供了预测TCR-抗原序列结合的功能。用户可以上传需要进行预测的TCR-抗原序列，网站将其传入模型端，并返回预测结果

4.Docking Station
Docking Station 用户可以上传PDB格式的TCR序列和抗原序列，网站利用ImmunoFlash Docking Station，将TCR与抗原的复杂相互作用转化为可视化的清晰图像

5.Download
Download 提供了构建和精炼ImmunoFlash模型所需的数据集，用户可随时下载

6.Example
Example 提供了预测TCR-抗原序列结合的示例文件，以及用于生成分子动画的PDB格式的TCR序列示例文件和抗原序列示例文件
